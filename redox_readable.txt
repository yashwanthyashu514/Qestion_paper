 ASSERTION--REASON / STATEMENT BASED QUESTIONS
\noindent**Read the statements carefully to mark the correct option out of the options given below**
\begin{enumerate}[label=(\arabic*), leftmargin=2em, itemsep=1pt]

Q. If both statements are true and Reason is the correct explanation of Assertion.

Q. If both statements are true but Reason is not the correct explanation of Assertion.

Q. If Assertion is true but Reason is false.

Q. If Assertion is false but Reason is true.
\end{enumerate}

Q. **Assertion:** In the reaction, $\frac{1}{2}O2 + F2 -> OF2$, Fluorine is oxidant.
**Reason:** Fluorine cannot show positive oxidation state.

Q. **Assertion:** In the reaction Zn(s) + Cu^{2+(aq) -> Zn^{2+}(aq) + Cu(s)}, Cu^{2+} ions act as oxidising agent and Zn atoms act as a reducing agent.
**Reason:** A substance which readily gain electrons from other substances is an oxidising agent while reducing agent is a substance which can lose electrons to other substances.

Q. **Assertion:** Zn(s) + Cu^{2+(aq) -> Zn^{2+}(aq) + Cu(s)} can be split into following half reactions:
Zn(s) -> Zn^{2+ + 2e-} (Oxidation half reaction)
Cu^{2+(aq) + 2e- -> Cu} (Reduction half reaction)
**Reason:** Every redox reaction can be split into two reactions, one representing loss of electrons and the other representing gain of electrons.

Q. **Assertion:** Oxidation number of carbon in H--C$\equiv$N is $+4$.
**Reason:** Carbon can show any oxidation number.

Q. **Assertion:** In NH4NO3, the oxidation number of the two N-atoms is not equal.
**Reason:** One N atom is present in the ammonium ion while the other is present in the nitrate ion.

Q. **Assertion:** Oxidation number of carbon in CH2O is zero.
**Reason:** CH2O (formaldehyde) is a covalent compound.

Q. **Assertion:** Nitrous acid (HNO2) may act as an oxidising agent as well as a reducing agent.
**Reason:** The oxidation number of nitrogen remains same in all the compounds.

Q. **Assertion:** Oxidation number of Ni in [Ni(CO)4] is zero.
**Reason:** Nickel is bonded to neutral ligand, carbonyl.
